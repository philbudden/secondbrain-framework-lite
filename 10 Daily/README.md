# 10 Daily

Use this folder for one Daily Note per working day.

## What Belongs Here

- Today's focus.
- Personal or professional to-do items.
- Notes and activity from the day.
- Decisions made today.
- Open questions that need follow-up.
- References to Context Packs or documents updated today.

## What Not To Put Here

- Full project histories.
- Long meeting transcripts.
- Complete policy or strategy documents.
- Permanent workstream context that should live in a Context Pack.

## How To Use With An LLM Chat

Upload yesterday's Daily Note when using `30 Skills/daily-note-rollover.md`. Upload today's Daily Note when using meeting or action skills if you want the LLM to return a complete updated Daily Note.

When a Daily Note is updated, save the complete returned Markdown document in this folder using the date format `YYYY-MM-DD.md`.
