# 00 Inbox

Use this folder for material that has arrived but has not yet been turned into usable working context.

## What Belongs Here

- Rough notes copied from a meeting chat.
- A document you need to process later.
- A quick thought that does not yet belong in a Daily Note or Context Pack.
- A temporary dump of actions before you decide where they belong.

## What Not To Put Here

- Finished Context Packs.
- Daily Notes.
- Long-term reference material that is no longer active.
- Anything sensitive that should not be uploaded into an LLM chat.

## How To Use With An LLM Chat

If something in the inbox needs processing, upload only that item and any relevant Context Pack. Use a skill from `30 Skills/` if one fits. Ask the LLM to return complete replacement documents for the notes that should be updated.

After processing, move the original inbox item to `90 Archive/` if you still need a record, or delete it if your organisation's retention rules allow that.
