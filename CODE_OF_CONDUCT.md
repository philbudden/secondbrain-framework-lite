# Code of Conduct

Be respectful, constructive, and considerate. Discuss ideas, prompts, templates, and documentation without attacking people.

Harassment, discrimination, disclosure of private information, and deliberately unsafe contributions are not acceptable.

This project is designed for personal and work knowledge management, so privacy matters. Do not post real private notes, meeting transcripts, credentials, customer information, employee information, or confidential work material in public discussions, examples, issues, or pull requests.

Maintainers may edit, close, or remove contributions and participation that violate these expectations. Serious or sensitive concerns should be reported privately through the repository's security reporting channel.
