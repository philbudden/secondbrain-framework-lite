# 20 Context Packs

Context Packs are compact, durable summaries of ongoing workstreams. They are written for two readers: you, and an LLM chat that needs enough context to help without seeing your whole vault.

## What Belongs Here

- One Context Pack per active workstream, project, recurring meeting, or responsibility area.
- Current purpose, status, key people, decisions, risks, next actions, and open questions.
- Links or filenames for important source documents.
- Short summaries of important background that future chats will need.

## What Not To Put Here

- Raw transcripts.
- Every daily note detail.
- Drafting scratch.
- Large copied documents.
- Sensitive material that should not be uploaded into chat.

## How To Use With An LLM Chat

Upload the relevant Context Pack whenever you ask the LLM for meeting prep, minutes, action extraction, status review, decision support, or drafting help about that workstream.

If the LLM returns an updated Context Pack, review it, then save the complete replacement Markdown document back into this folder using the same filename.
