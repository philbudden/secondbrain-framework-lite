# 90 Archive

Use this folder for material that is no longer active but may still be useful for reference.

## What Belongs Here

- Old inbox items that have been processed.
- Superseded notes kept for audit or reference.
- Closed workstream Context Packs.
- Meeting material that no longer needs to sit in active folders.

## What Not To Put Here

- Active Daily Notes.
- Current Context Packs.
- Reusable skills.
- Anything that should be deleted under your organisation's retention or data-handling rules.

## How To Use With An LLM Chat

Archive material is usually not uploaded. Only upload archived documents when they are directly relevant to the current question and you are comfortable sharing that content with the LLM service you are using.
